# HEIC Converter

A lightweight desktop application that converts `.heic` image files into widely supported formats such as `.jpg` and `.png`.

This tool is designed to be simple and easy to use — **no installation, coding, or setup required**.

---

## Overview

HEIC (High Efficiency Image Container) is the default image format used by Apple devices. While it offers better compression and quality, it is not supported by many platforms and applications.

This converter allows you to quickly convert HEIC images into formats that work everywhere.

---

## Features

- Convert HEIC to JPG or PNG
- Simple user interface
- No installation required
- Fast local processing (no internet needed)
- Supports images from iPhones and other Apple devices

### Step 2: Run the Application

---

### Step 3: Convert Your Files

1. Open the HEIC Converter
2. Drag and drop a `.heic` file into the app **or** click to select a file
3. Choose your output format (`.jpg` or `.png`)
4. Click **Convert**
5. Your converted image will be saved to your selected location

## Common Issues

### 1. App won't open

- Right-click the `.exe` file and select **Run as Administrator**
- Make sure Windows Defender or antivirus is not blocking the app

---

### 2. "Windows protected your PC" warning

This is normal for unsigned applications.

To proceed:
1. Click **More Info**
2. Click **Run Anyway**

---

### 3. File does not convert

- Make sure the file is a valid `.heic` image
- Try another HEIC file to confirm

---

### 4. Converted file not found

- Check your **Downloads** folder or the app’s default output directory
- Try selecting a specific save location before converting
